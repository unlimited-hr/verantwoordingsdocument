## Tools
Omdat het team in TLE 4 is begonnen aan het concept uitwerken, hadden we een plek nodig om alle code op te slaan wat ook toegankelijk was voor iedereen. Zoals we zijn aangeleerd, besloten we om GitHub te gebruiken. 

Op GitHub heb je de mogelijkheid om een ‘team’ aan te maken. In zo’n team kun je meerdere code repositories aanmaken
![Github team](./images/github.PNG)


Zoals je kunt zien staan alle repositories hierin en zijn deze toegankelijk voor alle teamleden.  

Ook deze periode hebben we weer gebruikt gemaakt van Jira voor onze planning. De volledige tijd dat we Jira hebben gebruikt, hebben we maar 1 opmerking over het programma. Dit is dat het fijn zou zijn om meerdere mensen aan 1 taak te koppelen. Verder hebben we Jira gebruikt zonder problemen. 

Ook Miro is deze sprint weer in gebruik genomen, maar minder dan vorige periode. Op Miro was minder te doen omdat we dit platform vooral gebruikten om onderzoeken vast te leggen. Wel werd Miro nogsteeds gebruikt voor feedbackmomenten.
